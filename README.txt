This is a light standalone distribution of termbox with static C version only
available and using make to build sources (original termbox repo contains
python version and uses waf build system)

Build:
    
    build termbox and create include/ and lib/
    $ make termbox
    build demos
    $ make demo
    clean everything
    $ make clean

Docs:
    
    Go to https://github.com/nsf/termbox to read docs and view other
    languages bindings.
